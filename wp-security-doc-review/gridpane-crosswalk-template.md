# Vendor Research Crosswalk Template

Use this template when a vendor-specific research note needs to be translated into editorial follow-up work.

## Rules

- Cite the exact public source URL for every vendor-specific claim.
- Distinguish "vendor claim" from "transferable editorial pattern."
- Point follow-up work at canonical source docs, not archived review artifacts in this repo.
- Mark proprietary terminology explicitly.
